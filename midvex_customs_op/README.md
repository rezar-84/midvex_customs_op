# Customs Operations

Customs Operations is an Odoo 19 module designed to manage international shipments, track required compliance documents, verify originals, and automate readiness calculations.

## Installation

1. Place this directory inside your Odoo addons path.
2. Enable developer mode.
3. Update the App list.
4. Search for `Customs Operations` and click Install.

## Configuration

* **Stages**: Define stages of shipment progress under Configuration > Stages.
* **Document Types**: Define document types (like Certificate of Analysis, Health Certificate) under Configuration > Document Types.
