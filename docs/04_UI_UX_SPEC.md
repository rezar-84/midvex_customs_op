# UI/UX Specification

## UX objective

A purchasing or logistics user should be able to view, trigger, and track compliance status, transport status, and bills directly from their natural environment (Purchase Orders, stock receipts, and bills) and drill down into the Customs File.

---

## 1. Purchase Order Form Extension

### Header Actions
- **Create Import Operation** (Button): Visible in draft or confirmed state if no Customs File is linked. Creates a new `customs.operation` and links it.
- **Open Import Operation** (Button): Visible if a Customs File is already linked. Redirects directly to the form view of the linked operation.
- **Sync Customs Operation** (Button): Syncs lines, vendor info, and expected dates.

### Smart Buttons (Upper Right)
- **Import / Customs Operations** (Smart Button): Displays a shopping-cart/cargo icon, the count of linked operations (`customs_operation_count`), and opens the linked list/kanban when clicked.

### Customs Info Box / Section
Add a dedicated group or tab "Import & Customs" showing:
- `customs_required` (Boolean Toggle): Manual flag to force customs.
- `is_import_purchase` (Boolean, Read-only): Automatically computed based on vendor country.
- `customs_operation_state` (Badge widget): Displays current stage of the linked Customs File (e.g. "Waiting for Documents", "Customs Cleared").
- `customs_document_completion_percentage` (Progressbar widget): Completion rate of required documents.
- `customs_shipment_ready` (Boolean / Ribbon): Indicates readiness.
- `customs_blocking_reason_text` (Text, Red/Warning style): Displayed if blocked, explaining why.

---

## 2. Stock Picking Form Extension

### Smart Buttons
- **Customs File** (Smart Button): Displays a folder or warning icon, and navigates to the linked `customs.operation` record.

### Form Fields & Warnings
- **Customs Stage Summary** (Char, Read-only): Displayed in the header/chatter section, showing linked operation's stage (e.g. "Customs Clearance", "On Hold").
- **Clearance Alert / Banner**: If the picking is a receipt generated by an import PO and is not cleared, display a soft warning message widget: *"Goods are being received before Customs Operation is cleared."*

---

## 3. Account Move (Vendor Bill) Form Extension

### Smart Buttons
- **Customs Operations** (Smart Button): Displays a folder or dollar-bill icon, showing linked Customs Files and allowing direct drill-down.

---

## 4. Customs File Form (`customs.operation`)

### Smart Buttons
- **Purchase Orders** (Smart Button): Links back to the source POs.
- **Stock Receipts** (Smart Button): Links to linked stock pickings.
- **Vendor Bills** (Smart Button): Links to linked vendor bills (both PO-generated bills and linked expense bills).
- **Sales Orders** (Smart Button): Links to linked sales orders (if MTO/Procured for customer).

### Tabs
1. **Overview**: Links to POs, receipts, suppliers, broker, forwarder, carrier, currency, and commercial value summaries.
2. **Product Lines**: Table of product lines linked to `purchase.order.line` with quantities, batch, and compliance requirements.
3. **Required Documents**: List of required documents (completion status, upload widgets).
4. **Shipment Details**: Departure, arrival, country routing, and tracking codes.
5. **Customs & Laboratory**: Declaration number, release date, and inspection/lab dates.
6. **Notes**: Text notes, manager override reasons, and warnings.

---

## 5. Visual States

- **Red**: Rejected, expired, or overdue mandatory document.
- **Orange**: Correction required, approaching deadline, or stock picking received before clearance (soft warning).
- **Green**: Approved, accepted, or complete readiness.
- **Muted**: Not applicable stage/field.

Ensure status is supported by text labels or badges, not just color codes (WCAG accessibility compliance).

---

## 6. End-User Review Checklist

For each screen, verify:
- Can a PO manager see if the customs process has started without leaving the PO form?
- Is it clear to logistics why a shipment is blocked?
- Does the warehouse team get alerted if they receive uncleared goods?
- Can accounts see which bills belong to which customs file?
- Are company boundaries respected across all linked documents?
