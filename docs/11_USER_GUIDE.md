# User Guide

## Creating a Customs File

1. Open Customs Operations.
2. Select Operations.
3. Create a new Customs File.
4. Add responsible employee, supplier, broker, and shipment information.
5. Link the purchase order.
6. Add shipment product lines.
7. Save.

## Adding required documents

1. Open the Documents tab.
2. Add each required document.
3. Select requirement level.
4. Assign vendor and responsible user.
5. Set deadline.
6. Mark whether an original is required.

## Reviewing documents

1. Open the document requirement.
2. Upload the received file.
3. Mark it Under Review.
4. Approve it, reject it, or request correction.
5. Enter a reason when rejecting or requesting correction.

## Tracking originals

Record:

- Original issued
- Original dispatched
- Courier
- Tracking number
- Dispatch date
- Original received

## Checking readiness

Review:

- Completion percentage
- Missing documents
- Rejected documents
- Expired documents
- Original-document issues
- Blocking reasons

Resolve every blocking reason before the shipment departs.

## Closing a Customs File

Record:

- Customs declaration
- Customs release
- Warehouse delivery

Close the file only after all mandatory operational information is complete.
