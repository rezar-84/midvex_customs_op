# Milestone Execution Prompt

Implement the requested milestone from `docs/09_IMPLEMENTATION_PLAN.md`.

## Before coding

- Restate the milestone scope.
- Restate its acceptance criteria.
- List files expected to change.
- Confirm no unrelated files will be modified.
- Identify dependencies and risks.

## During implementation

- Follow `AGENTS.md`.
- Follow Odoo 19 conventions.
- Update `docs/10_DECISIONS.md` for significant decisions.
- Add tests with each feature.
- Maintain multi-company and security correctness.
- Do not weaken rules to make tests pass.

## After implementation

- Run installation or upgrade.
- Run relevant tests.
- Review logs.
- Review access rights.
- Review the UI as every affected role.
- Update `docs/09_IMPLEMENTATION_PLAN.md`.
- Update `CHANGELOG.md`.
- Produce a concise implementation report.
- Stop before beginning the next milestone.
